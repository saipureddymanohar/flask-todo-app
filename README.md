# 📝 Flask To-Do List App

A simple web-based To-Do list application built with Flask. Users can add, mark as done, and delete tasks in real-time.

## 🚀 Features
- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- In-memory storage (no database)

## 📦 Requirements
- Python 3.x
- Flask
